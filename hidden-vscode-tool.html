Step-by-Step Guide

1. Install Extension:
VS Code → Extensions → Search "Cline-AI" → Install.

2. Open Cline Panel:
Click the robot icon in the sidebar.

3. Select API Provider:
Choose Google Gemini.

4. Get API Key:
Go to Google AI Studio → Create key → Copy it.

5. Configure Model:
Paste key and select gemini-2.0-flash-thinking-exp-1219.

6. Start Coding:
Try prompts like "understand the codebase".
